## 环境准备

在开始使用智演助手之前，您需要准备以下环境：

### 1. 安装Python

确保您的系统已安装Python 3.7或更高版本。

检查Python版本：
```bash
python --version
```

如果未安装Python，请从官网下载并安装：https://www.python.org/downloads/

### 2. 安装Git（可选）

如果您想使用Git管理项目，可以安装Git。

下载地址：https://git-scm.com/downloads

## 安装步骤

### 1. 获取项目文件

如果您已经有项目文件，可以跳过此步骤。

### 2. 安装Python依赖

打开命令行终端，进入项目目录：

```bash
cd d:\PYTHON\simple_webpage
```

安装所需的Python包：

```bash
pip install -r requirements.txt
```

如果安装速度较慢，可以使用国内镜像源：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

## 配置API密钥

智演助手需要配置AI视频生成模型的API密钥才能正常工作。目前支持以下三种模型：

1. Seedance（字节跳动）
2. RunwayML Gen-2（国内版）
3. Pika Labs（国内版）

### 选择一个模型进行配置

您可以选择其中一个或多个模型进行配置。下面以Seedance为例：

#### 配置Seedance API密钥

1. 访问火山引擎：https://www.volcengine.com/
2. 注册/登录火山引擎账号
3. 进入"火山引擎控制台"
4. 选择"模型推理"服务
5. 创建推理端点：
   - 点击"创建推理端点"按钮
   - 选择模型：Seedance
   - 填写端点名称和描述
   - 点击"确定"创建
6. 获取API Key和端点ID

#### 在app.py中配置API密钥

打开`app.py`文件，找到以下部分：

```python
AI_VIDEO_MODELS = {
    'seedance': {
        'name': 'Seedance (字节跳动)',
        'api_url': 'https://ark.cn-beijing.volces.com/api/v3/chat/completions',
        'api_key': os.environ.get('SEEDANCE_API_KEY', 'YOUR_SEEDANCE_API_KEY'),
        'model': 'ep-20240601111111-xxxxx',  # 需要替换为实际的端点ID
        'description': '字节跳动旗下的AI视频生成引擎'
    },
    # ... 其他模型配置
}
```

将`YOUR_SEEDANCE_API_KEY`替换为您从火山引擎获取的API密钥，将`ep-20240601111111-xxxxx`替换为实际的端点ID。

例如：

```python
'seedance': {
    'name': 'Seedance (字节跳动)',
    'api_url': 'https://ark.cn-beijing.volces.com/api/v3/chat/completions',
    'api_key': 'your_actual_api_key_here',
    'model': 'ep-20240601111111-abc123',
    'description': '字节跳动旗下的AI视频生成引擎'
},
```

#### 配置其他模型

如果您想配置其他模型，请参考`API_KEY_GUIDE.md`文件中的详细说明。

## 启动服务

### 1. 启动后端服务

在命令行终端中，进入项目目录：

```bash
cd d:\PYTHON\simple_webpage
```

启动Flask后端服务：

```bash
python app.py
```

您应该会看到类似以下的输出：

```
 * Serving Flask app 'app'
 * Debug mode: on
 * Running on http://0.0.0.0:5000
```

这表示后端服务已成功启动，运行在`http://localhost:5000`。

### 2. 访问前端页面

有两种方式访问前端页面：

#### 方法一：直接打开

双击`index.html`文件，用浏览器打开。

#### 方法二：使用Python内置服务器

打开一个新的命令行终端，进入项目目录：

```bash
cd d:\PYTHON\simple_webpage
```

启动Python内置HTTP服务器：

```bash
python -m http.server 8000
```

然后在浏览器中访问：http://localhost:8000

## 使用网页生成视频

### 1. 上传文档

1. 在网页中找到"上传您的文档"部分
2. 点击上传区域或拖拽文件到上传区域
3. 选择您要上传的文档（支持PDF、PPT、PPTX、DOC、DOCX、TXT格式）
4. 上传成功后，会显示文件信息

### 2. 选择AI视频生成模型

1. 在"AI视频生成模型选择"下拉菜单中，选择您配置的模型
   - Seedance（字节跳动）
   - RunwayML Gen-2（国内版）
   - Pika Labs（国内版）

### 3. 设置视频参数

1. 选择视频风格：
   - 商务风格
   - 创意风格
   - 简约风格
   - 科技风格

2. 选择视频时长：
   - 短（1-2分钟）
   - 中（3-5分钟）
   - 长（5-10分钟）

### 4. 生成视频

1. 点击"生成视频"按钮
2. 等待视频生成完成，您会看到进度条显示当前进度
3. 生成完成后，会显示视频预览

### 5. 预览和下载视频

1. 在视频预览区域，您可以观看生成的视频
2. 点击"下载视频"按钮，将视频保存到本地

## 常见问题

### Q1: 启动后端服务时出现错误

**问题**：运行`python app.py`时出现错误

**解决方案**：
1. 确保已安装所有依赖：`pip install -r requirements.txt`
2. 检查Python版本是否为3.7或更高
3. 确保没有其他程序占用5000端口

### Q2: 上传文件时出现错误

**问题**：上传文件时提示错误

**解决方案**：
1. 确保后端服务已启动
2. 检查文件格式是否支持（PDF、PPT、PPTX、DOC、DOCX、TXT）
3. 检查文件大小是否过大

### Q3: 生成视频时出现错误

**问题**：点击"生成视频"按钮后提示错误

**解决方案**：
1. 确保已正确配置API密钥
2. 检查API密钥是否有效
3. 检查网络连接是否正常
4. 查看后端控制台输出的错误信息

### Q4: 视频生成时间过长

**问题**：视频生成需要很长时间

**解决方案**：
1. 视频生成时间取决于文档大小和AI模型响应速度
2. 可以尝试选择较短的时长
3. 可以尝试不同的AI模型

### Q5: 无法下载视频

**问题**：点击"下载视频"按钮后无法下载

**解决方案**：
1. 确保视频已成功生成
2. 检查浏览器是否允许下载
3. 尝试使用不同的浏览器

## 高级使用

### 使用环境变量存储API密钥

为了安全起见，建议使用环境变量存储API密钥：

1. 创建一个`.env`文件：

```
SEEDANCE_API_KEY=your_actual_api_key_here
RUNWAY_API_KEY=your_actual_api_key_here
PIKA_API_KEY=your_actual_api_key_here
```

2. 安装python-dotenv：

```bash
pip install python-dotenv
```

3. 在app.py开头添加：

```python
from dotenv import load_dotenv
load_dotenv()
```

### 自定义视频风格和时长

您可以根据需要修改app.py中的视频风格和时长设置：

```python
# 根据不同的视频时长设置秒数
duration_map = {
    'short': 60,    # 1-2分钟
    'medium': 180,  # 3-5分钟
    'long': 360     # 5-10分钟
}

# 根据不同的视频风格设置提示词
style_prompts = {
    'business': '商务专业风格，简洁大气',
    'creative': '创意风格，色彩鲜明，富有想象力',
    'minimalist': '简约风格，干净整洁，重点突出',
    'tech': '科技风格，未来感，蓝色调'
}
```
