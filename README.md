# 🎬 智演助手 — AI 视频生成平台

> 上传文档 → AI 自动生成视频。6 步全自动，从剧本到成片一气呵成。

## 它能做什么

你只需上传一份文档（.txt / .docx），系统自动完成：

```
文档解析 → 分镜设计 → 图片生成 → 视频生成 → 配音字幕 → 合成输出
```

## 用到的 AI 模型

| 步骤 | 模型 |
|------|------|
| 理解内容 + 写分镜 | DeepSeek V4 Pro |
| 逐镜生成图片 | Seedream 5.0 Lite |
| 图片变视频片段 | Seedance 2.0 Fast |
| 旁白配音 | Edge TTS |
| 视频拼接合成 | ffmpeg |

## 快速开始

```bash
# 1. 安装
pip install -r requirements.txt

# 2. 配置密钥
cp .env.example .env   # 编辑填入 API Key

# 3. 启动
python app.py           # 浏览器打开 http://localhost:5000
```

需要准备的 API Key：

- **火山引擎 ARK** — Seedream + Seedance（图生图/图生视频）
- **DeepSeek** — 内容理解与分镜设计

## 使用说明

启动后浏览器打开 `http://localhost:5000`，按以下步骤操作：

### 1. 上传文档

拖拽或点击上传 `.txt` 或 `.docx` 文件（最大 100MB）。系统会自动解析文档内容，提取核心信息。

支持的格式：`.txt` `.docx`

### 2. 选择参数

| 参数 | 选项 |
|------|------|
| 语言 | 中文 / English / 日本語 / 한국어 |
| 视觉风格 | 3D 卡通 / 2D 扁平 / 写实简化 / 素描线稿 |
| 时长 | AI 自动分配 / 1 分钟 / 3 分钟 / 5 分钟 |
| 分辨率 | 1920×1080（横屏）/ 1024×1024（方形）/ 1080×1920（竖屏） |

### 3. 一键生成

点击「开始生成」，系统自动执行 6 步流水线，每步进度实时显示：

1. 🔍 **文档解析** — 提取全文内容
2. 📝 **分镜设计** — LLM 编排镜号、场景、时长
3. 🎨 **图片生成** — 逐镜出图
4. 🎥 **视频生成** — 图片驱动视频片段（支持推近/拉远/跟拍等 8 种运镜）
5. 🎙️ **配音字幕** — 自动生成旁白 + 叠加字幕
6. 🎬 **合成输出** — 拼接所有片段，输出完整视频

### 4. 下载视频

生成完成后，点击下载按钮保存到本地。
