# 简易网页项目

这是一个简单而美观的响应式网页项目，使用纯HTML、CSS和JavaScript构建。

## 项目结构

```
simple_webpage/
├── index.html      # 主HTML文件
├── styles.css      # 样式表
├── script.js       # JavaScript交互脚本
└── README.md       # 项目说明文档
```

## 功能特点

- 响应式设计，适配各种设备尺寸
- 现代化UI设计，包含导航栏、英雄区域、关于部分和联系表单
- 平滑滚动效果
- 表单验证和提交处理
- 特性卡片动画效果
- 导航栏滚动效果

## 如何使用

1. 直接在浏览器中打开 `index.html` 文件即可查看网页
2. 或者使用本地服务器（如Live Server）来运行项目

## 自定义修改

- 修改 `index.html` 来调整页面内容和结构
- 修改 `styles.css` 来自定义网页样式
- 修改 `script.js` 来调整交互功能

## 浏览器兼容性

本项目支持所有现代浏览器，包括：
- Chrome
- Firefox
- Safari
- Edge
