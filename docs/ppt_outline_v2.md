## 设计风格

| 要素 | 选择 |
|------|------|
| 配色 | 深黑(#0D0D0D)主背景 + 金色(#D4AF37)强调 + 白色文字 |
| 字体 | Arial Black 标题 + Microsoft YaHei 正文 |
| 图标 | 每页至少1个视觉焦点(截图/图表/数字/图标)，绝不用纯文字页 |
| 数据展示 | 大数字(60-72pt) + 小标签 |
